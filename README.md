# 🩸 Fingerprint Blood Group Prediction

A Machine Learning based application that predicts a person's blood group using fingerprint images. This project aims to explore the relationship between fingerprint patterns and blood groups by training a classification model on fingerprint datasets.

## 📌 Overview

The Fingerprint Blood Group Prediction System uses image processing and machine learning techniques to classify fingerprint images into different blood groups.

The project includes:
- Fingerprint image preprocessing
- Feature extraction
- Model training
- Blood group prediction
- User-friendly interface for testing new fingerprints

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/yourusername/Fingerprint-Blood-Group.git
```

### Navigate to project

```bash
cd Fingerprint-Blood-Group
```

### Install dependencies

```bash
pip install -r requirements.txt
```
